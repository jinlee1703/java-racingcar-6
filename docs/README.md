# 자동차 경주

## 기능 목록

### 1. 사용자 입력받기

- [x] 자동차 이름 입력
    - [x] 자동차 이름은 쉼표(,)로 구분
    - [x] 빈 라인 입력 불가
- [x] 시도할 횟수 입력
    - [x] 정수만 입력 가능
    - [x] 빈 라인 입력 불가
- [ ] 사용자가 잘못된 값을 입력했을 경우 `IllegalArguemntException` 발생시키기
    - 자동차 이름 예외
        - [ ] 자동차 이름을 입력하지 않았을 경우
    - 시도 횟수 예외
        - [ ] 시도할 횟수를 입력하지 않았을 경우
        - [ ] 시도할 횟수를 정수로 입력하지 않았을 경우

### 2. 결과 출력하기

- [ ] 각 차수별 실행 결과 출력하기
    - [ ] 각 자동차 이름과 실행 결과 출력
    - [ ] 컴퓨터 전진 여부 판단을 위한 난수 생성
    - [ ] 컴퓨터 전진 여부 판단을 위한 조건 설정
- [ ] 최종 우승자 출력하기
    - [ ] 여러 명일 경우 쉼표(,)로 구분하여 출력하기
